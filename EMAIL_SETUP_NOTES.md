# メール設定に関するメモ

## ユーザー招待機能について

### 現状
- Supabaseのメールテンプレートに「Invite user」テンプレートを設定済み
- アプリの性質上、現時点ではユーザー招待機能は使用していない

### 今後の検討事項
- **マーケティング用途**: 将来的にマーケティング目的でユーザー招待機能を使用する可能性がある
- **テンプレートの更新**: マーケティング用途で使用する場合は、招待メールの内容を変更する必要がある
  - 現在のテンプレートは一般的な招待メール形式
  - マーケティング用途では、キャンペーン情報や特典などの追加が必要になる可能性がある

### 対応が必要な場合
1. 招待メールテンプレートの内容を確認・更新
2. マーケティング用の文言やデザインに変更
3. キャンペーン情報や特典情報の追加
4. 招待リンクの有効期限や条件の確認
